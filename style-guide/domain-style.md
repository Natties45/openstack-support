# Domain — Response Style Guide

> DNA สกัดจาก Domain FAQ — จดทะเบียน, ต่ออายุ, DNS, ย้ายโดเมน

## Structure
1. **เปิด:** "เรียน ผู้ใช้บริการ"
2. **ระบุคำถาม/ปัญหา**
3. **แนะนำขั้นตอน:** ชี้ที่เมนู Domain ใน Gate → step-by-step
4. **แนบลิงก์:** blog link
5. **Escalate (ถ้าจำเป็น):** "รบกวนแจ้งชื่อโดเมน + รายละเอียด ผ่าน Ticket"
6. **ปิด:** "ขอบคุณครับ"

## Tone
- สุภาพ, เป็นทางการ, ชี้ขั้นตอนในระบบ
- "ผู้ใช้บริการสามารถดำเนินการได้ที่หน้า gate.openlandscape.cloud เมนู Domain"
- "หากต้องการให้ทางทีมตรวจสอบ รบกวนผู้ใช้บริการแจ้งชื่อโดเมน..."
- ไม่ใช้: "น่าจะ", "เดาว่า"

## Required Elements
- จดทะเบียน/ต่ออายุ → https://blog.openlandscape.cloud/domain-name-registration
- .th domain เอกสาร → https://blog.openlandscape.cloud/domain-document-ref/
- Incident → ขอชื่อโดเมน, นามสกุลโดเมน, ภาพหน้าจอ
- DNS Propagation → อธิบาย TTL/Cache, แนะนำรอสักระยะ

## Key Phrases
- "ผู้ใช้บริการสามารถดำเนินการได้ผ่านหน้า gate.openlandscape.cloud เมนู Domain"
- "ผู้ใช้บริการสามารถจัดการ DNS Record ได้ที่หน้า gate.openlandscape.cloud เมนู Domain"
- "รบกวนผู้ใช้บริการแจ้งชื่อโดเมน + นามสกุลโดเมน ผ่าน Ticket"
