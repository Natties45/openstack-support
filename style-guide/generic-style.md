# Generic Replies — Response Style Guide

> DNA สกัดจาก Template ตอบกลับทั่วไป (รับทราบ, แจ้งผล, ติดตาม, ปิด Ticket)

## Structure
1. **เปิด:** "เรียน ผู้ใช้บริการ"
2. **ความคืบหน้า:** รับทราบ / กำลังตรวจสอบ / ดำเนินการแล้ว
3. **Follow-up:** ถามว่ายังมีปัญหาไหม (ถ้าติดตาม)
4. **ช่องทางติดต่อ:** "สามารถติดต่อผ่าน Ticket Support ได้ตลอดเวลา"
5. **ปิด:** "ขอบคุณครับ"

## Tone
- สุภาพ, อ่อนโยน, ไม่ technical
- "ทาง OpenLandscape รับทราบข้อมูลและกำลังดำเนินการตรวจสอบ"
- "ทาง Openlandscape ดำเนินการแก้ไขเรียบร้อยแล้วครับ"
- "Openlandscape ยินดีให้บริการครับ"

## Variants
1. **Acknowledge:** รับทราบ → กำลังตรวจสอบ
2. **Done:** ดำเนินการแล้ว → รบกวนตรวจสอบ
3. **Follow-up:** ยังพบปัญหาไหม → ติดต่อได้ตลอด
4. **Need Info:** ขอเบอร์ติดต่อเพิ่ม
5. **Callback:** เจ้าหน้าที่จะติดต่อกลับ

## Ticket Reply Mode (ในระบบ internal.openlandscape.cloud)

| สถานะ | รูปแบบ |
|---|---|
| **Reply แรก** | ใช้ "เรียน ผู้ใช้บริการ" + "ขอบคุณครับ" |
| **Reply ถัดไป** | **ไม่ต้อง** ขึ้น/ปิด formal — ตอบตรงประเด็น |
| **Reply แจ้ง progress** | ปิดด้วย "ครับ/ค่ะ — หากมีความคืบหน้าจะแจ้งให้ทราบอีกครั้งครับ" |
| **Gender** | "ครับ"/"ค่ะ" ตามเพศผู้ตอบ |

> ใน Ticket system มี Timestamp + sender อยู่แล้ว
