# วิธีตรวจสอบการใช้งาน Utilize ของ Server สำหรับ Windows Server และ Linux

สำหรับ Windows Server สามารถตรวจสอบการทำงานบนระบบปฏิบัติการได้ โดยการใช้ Task Manager ซึ่งเป็นโปรแกรมสำหรับแสดงข้อมูลการทำงานของคอมพิวเตอร์ที่ใช้งาน ณ ปัจจุบัน โดยมีขั้นตอนดังต่อไปนี้

1. ค้นหาโปรแกรม Task Manager

2. จากนั้นทำการกด More details เพื่อดูข้อมูลเพิ่มเติม

3. จะปรากฏหน้าต่าง Processes Tab แสดงข้อมูลในส่วนของ CPU, Memory, Disk, Network และ GPU

4. นอกจากนี้ยังสามารถดูในส่วนของ Performance Tab ซึ่งจะแสดงข้อมูลโดยรวม ของ CPU, Memory, Disk, Wi-Fi และ GPU ได้เช่นกัน

## วิธีตรวจสอบการใช้งาน Utilize ของ Server สำหรับ Linux

สำหรับ Linux Server สามารถตรวจสอบการทำงานบนระบบปฏิบัติการได้ โดยการใช้คำสั่ง (Command Line) ในการตรวจสอบ มีวิธีการดังต่อไปนี้

### ตรวจสอบ CPU
- คำสั่ง `top` — ใช้สำหรับตรวจสอบการทำงานโดยรวมของ CPU
- คำสั่ง `lscpu` — ใช้สำหรับตรวจสอบข้อมูล CPU

### ตรวจสอบ Memory
- คำสั่ง `free -m` — ใช้สำหรับตรวจสอบหน่วยความจำ

### ตรวจสอบ Disk
- คำสั่ง `df -h` — ใช้สำหรับตรวจสอบพื้นที่จัดเก็บข้อมูล

### ตรวจสอบ Network
- คำสั่ง `iftop` — ใช้สำหรับตรวจสอบการใช้งาน Network (ต้องติดตั้งเพิ่มเติม)
- คำสั่ง `nload` — ใช้สำหรับตรวจสอบการใช้งาน Network ทั้งขาเข้าและขาออก

---

## หากพบว่า Utilize สูงผิดปกติ

หากพบว่ามีการใช้งาน CPU หรือ Memory สูงผิดปกติ แนะนำให้ตรวจสอบ:
1. มีโปรแกรมหรือ Service ใดที่ใช้ทรัพยากรมากผิดปกติ
2. ตรวจสอบว่ามีการโจมตีหรือไม่ (เช่น Malware, Brute Force)
3. หากจำเป็น สามารถทำการ Resize Instance เพื่อเพิ่มทรัพยากร
