# Generic Response — Style Guide

## Universal Rules (From AGENTS.md)
- Open with: "เรียน ผู้ใช้บริการ"
- Close with: "ขอบคุณครับ"
- Polite, formal Thai
- Technical terms in English
- Use "ครับ" not "ค่ะ"

## Acknowledgment
- "เรียน ผู้ใช้บริการ ทาง OpenLandscape ได้รับเรื่องของท่านเรียบร้อยแล้ว ขณะนี้อยู่ระหว่างดำเนินการตรวจสอบ"
- Include ticket reference if available

## Resolution
- "เรียน ผู้ใช้บริการ ทางทีมได้ดำเนินการแก้ไขเรียบร้อยแล้ว"
- Summarize what was done
- Ask customer to verify

## No Response Follow-up
Template 1 (3 days): "เรียน ผู้ใช้บริการ ทางทีมขอติดตามผลการดำเนินการ..."
Template 2 (7 days): "เรียน ผู้ใช้บริการ เนื่องจากยังไม่ได้รับการตอบกลับ ทางทีมขอปิด Ticket นี้ หากยังมีข้อสงสัยสามารถเปิด Ticket ใหม่ได้"

## Callback Notification
- "เรียน ผู้ใช้บริการ ทางเจ้าหน้าที่จะติดต่อกลับภายใน..."
- Specify channel (โทรศัพท์ / อีเมล / Ticket)

## Need More Info
- "เพื่อความรวดเร็วในการตรวจสอบ รบกวนแนบ..."
- List what is needed clearly

## Working Hours
- จันทร์–ศุกร์ 09:00–18:00 น.
- Outside: "ทางทีมจะตรวจสอบและอัปเดตผ่าน Ticket ในวันทำการถัดไป"
