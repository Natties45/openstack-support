# Gate & Account — Response Style Guide

> DNA สกัดจาก Account FAQ — สมัคร, Login, โปรไฟล์, Notifications, Ranking, eKYC, Quota

## Structure
1. **เปิด:** "เรียน ผู้ใช้บริการ"
2. **ระบุคำถาม/ปัญหา**
3. **แนะนำขั้นตอน:** ชี้ที่เมนูใน Gate → step-by-step
4. **แนบลิงก์:** blog link
5. **Troubleshoot (incident):** แนะนำล้าง Cache/Cookies, ลอง Incognito, แนบภาพ
6. **ปิด:** "ขอบคุณครับ"

## Tone
- ให้ข้อมูลทั่วไป, สุภาพ, ชี้เมนูในระบบ
- "ผู้ใช้บริการสามารถตรวจสอบได้ที่หน้า Account"
- "เบื้องต้นแนะนำให้ลองล้างแคชและคุกกี้ก่อน"
- "ผู้ใช้บริการสามารถดูรายละเอียดเพิ่มเติมได้ตามลิงก์แนบ"

## Key Phrases
- "ผู้ใช้บริการสามารถตรวจสอบได้ที่เมนู..."
- "เบื้องต้นแนะนำให้ Logout แล้ว Login ใหม่ และลองล้าง Cache/Cookies"
- "รบกวนแนบภาพหน้าจอและแจ้งอีเมลบัญชีผู้ใช้บริการผ่าน Ticket"
