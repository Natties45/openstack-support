# NOC / Call Center — Response Style Guide

> DNA สกัดจาก NOC Scripts และ Call Center templates

## Structure
1. **[NOC] header:** ระบุว่าเป็น NOC → OP หรือ NOC → Technical
2. **ข้อมูลผู้ใช้บริการ:** ชื่อ, เบอร์, อีเมล (ตัวแปร)
3. **รายละเอียด:** หัวข้อ, รายละเอียด
4. **การดำเนินการเบื้องต้น:** สิ่งที่ NOC ทำไปแล้ว
5. **ท้ายข้อความ:** แจ้งเวลาทำการ/นอกเวลา
6. **ปิด:** "ขอบคุณครับ NOC / Call Center OpenLandscape"

## Tone
- กระชับ, ส่งต่อ, บันทึกข้อมูล
- "ทาง NOC ขอส่งต่อข้อมูลจากผู้ใช้บริการ..."
- "ทางทีม NOC ได้เปิด Ticket และสรุปแนวทาง/ขั้นตอนไว้ใน Ticket แล้ว"

## Required Elements
- แจ้งเวลาทำการ: จันทร์–ศุกร์ 09:00–18:00 น.
- ถ้านอกเวลา: "ทางทีมจะอัปเดตผ่าน Ticket ในวันทำการถัดไป"

## Variants
- Customer + incident → open ticket, escalate to OP
- Customer + request → answer + close
- Non-customer → email reply
- Claim customer → verify gate → escalate to Technical
