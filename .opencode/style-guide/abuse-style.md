# Abuse — Style Guide

## Tone
- Firm, professional, non-negotiable
- Reference Terms & Conditions explicitly
- State consequences clearly (suspension period, permanent ban conditions)

## Response Structure
1. Identify violation type (brute force, phishing, spam, bucket abuse)
2. State detection method (automated monitoring, complaint received)
3. State required action by customer
4. State deadline
5. State consequences of non-compliance
6. Reference T&C

## Common Phrases
- "ทาง OpenLandscape ตรวจพบการกระทำที่เข้าข่ายผิดข้อกำหนด..."
- "ขอความร่วมมือดำเนินการแก้ไขภายใน [N] วัน"
- "หากไม่ดำเนินการ ทางบริษัทขอสงวนสิทธิ์ในการ..."
- "สามารถศึกษาข้อกำหนดเพิ่มเติมได้ที่..."

## Violation Types
- Brute Force / Unauthorized Access
- Phishing / Fraudulent Content
- Spam / Unsolicited Email
- Object Storage Malicious Content
- Repeat Offender (ภายใน 30 วัน)

## Policy Reference
Always include: https://openlandscape.cloud/policy/term_and_condition
