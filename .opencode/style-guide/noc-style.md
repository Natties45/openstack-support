# NOC / Call Center — Style Guide

## Tone
- Professional, efficient, script-driven
- For ticket creation: gather all info before confirming
- For claims: acknowledge concern, explain process, set expectations

## Ticket Creation Checklist
Before creating ticket, confirm:
- Customer name & contact
- Instance/service affected
- Issue description
- Urgency level
- Preferred contact method

## Callback Policy
- Callback within 2 hours during working hours
- Outside working hours: next business day
- Customer callback requests logged in ticket

## Escalation Templates
- Level 1 → Senior Support (technical issues)
- Level 2 → Engineering (infrastructure issues)
- Level 3 → Management (billing disputes, abuse appeals)

## Common Phrases
- "ทางทีมได้รับเรื่องแล้ว จะดำเนินการภายใน..."
- "Ticket ID ของท่านคือ [TICKET_ID]"
- "เจ้าหน้าที่จะติดต่อกลับผ่านช่องทางที่ท่านแจ้งไว้"
