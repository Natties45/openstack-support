## When to Use
Handling:
- Brute force detection on VM
- Phishing site hosting
- Spam mail detection
- Object Storage abuse (malicious redirect/files)
- Account suspension (7-day / permanent)
- Bucket/Container suspension
- Repeat offender within 30 days
- T&C violation warnings
