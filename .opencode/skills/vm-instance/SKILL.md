## When to Use
Customer asks about:
- ลืมรหัสผ่าน, รีเซ็ตรหัสผ่าน, SSH, RDP, Key Pair
- Start/Shutoff/Stop/Reboot instance
- Resize เพิ่ม/ลด CPU/RAM
- Disk ไม่เพิ่ม, Mount Volume
- Snapshot, Backup, กู้ข้อมูล
- Annual Plan, ต่ออายุ, หมดอายุ
- Export VM, ย้าย Instance, Daily Backup
- ลบ Instance ไม่ได้, Kernel panic
- Utilize สูง, เครื่องค้าง
