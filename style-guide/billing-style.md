# Billing & Payment — Response Style Guide

> DNA สกัดจาก Billing/Tax/WHT/Refund FAQ

## Structure
1. **เปิด:** "เรียน ผู้ใช้บริการ"
2. **ชี้แจงนโยบาย/ขั้นตอน**
3. **แจ้งยอด/อัตรา** (ถ้าเกี่ยวข้อง)
4. **ขั้นตอนดำเนินการ**
5. **แนบลิงก์:** blog link
6. **ปิด:** "ขอบคุณครับ"

## Tone
- เป็นทางการ, อ้างอิงนโยบาย, ระบุตัวเลขชัดเจน
- "ตามข้อตกลงและเงื่อนไขการให้บริการ..."
- "โดยทั่วไปบริษัทจะดำเนินการภายใน X วันทำการ"
- "รบกวนผู้ใช้บริการดำเนินการ..."

## Required Elements
- เติมเงิน → https://blog.openlandscape.cloud/top-up/
- Schedule Top-up → https://blog.openlandscape.cloud/schedule-top-up
- ใบเสนอราคา → https://blog.openlandscape.cloud/request-for-quotation/
- ใบกำกับภาษี → https://blog.openlandscape.cloud/e-tax
- WHT → แจ้ง % (นิติบุคคล 3%, ราชการ 1%), วิธีส่งเอกสาร
- Refund → อ้างอิง T&C https://openlandscape.cloud/policy/term_and_condition

## Key Phrases
- "รบกวนแนบสลิป/หลักฐานการชำระเงินผ่าน Ticket"
- "ผู้ใช้บริการไม่สามารถทำการขอคืนเงินสำหรับรายการที่ชำระสำเร็จแล้วได้"
- "ลูกค้านิติบุคคล/บริษัท สามารถหักภาษี ณ ที่จ่ายได้"
